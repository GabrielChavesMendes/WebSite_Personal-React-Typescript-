## 📖 Sobre o Projeto

Este projeto é uma landing page customizada para serviços de Personal Trainer, desenvolvida ao longo de 4 semanas. A ideia central foi unir dois universos que vêm crescendo muito juntos: o **público amante da cultura pop/geek** e a **rotina de treinos e saúde**.

A inspiração visual e temática veio da estética e mentalidade do personagem Toji Fushiguro (*Jujutsu Kaisen*), muito utilizada no nicho fitness para atrair a atenção do público e incentivar a prática esportiva. O resultado é um site chamativo, autêntico e focado em alta conversão e engajamento.

## 🎯 Objetivos e Aprendizados

O principal objetivo deste projeto foi servir como um laboratório prático para elevar minhas habilidades no Front-End. Durante o desenvolvimento, foquei em aplicar boas práticas e explorar novas tecnologias:

* 🧩 **Componentização:** Utilização do React para dividir a interface em pedaços menores e reutilizáveis, facilitando a leitura, a escalabilidade e a manutenção do código.
* 🛡️ **Prevenção de Erros:** Adoção do TypeScript para garantir um código fortemente tipado, evitando bugs inesperados e deixando a aplicação mais robusta.
* ⚡ **Estilização Ágil:** Uso do Tailwind CSS para acelerar a criação de um layout 100% responsivo e fiel ao design feito no Figma, mantendo a performance alta.
* 🪄 **Animações e Experiência (UX/UI):** Integração da biblioteca **GSAP** para criar animações fluidas e dinâmicas, o que foi um desafio recompensador e mudou completamente a experiência do usuário na página.

## 🛠️ Tecnologias Utilizadas

* **Design:** Figma
* **Front-End:** React.js
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS
* **Animações:** GSAP (GreenSock Animation Platform)
* **Deploy:** Vercel
